## chartinkbyvk

chartinkbyvk is a simple Interactive CLI to fetch union/intersect of multiple Chartink scans

## Installation

```bash
pip install git+https://github.com/vkdatta/chartinkbyvk.git
```
## Upgrade

Use only if you are sure that all dependencies are there:

```bash
pip install --upgrade --force-reinstall --no-deps git+https://github.com/vkdatta/termuxessentialsbyvk.git
```

else use to even install dependencies from the scratch:

```bash
pip install --upgrade --force-reinstall git+https://github.com/vkdatta/termuxessentialsbyvk.git
```

else use:

```bash
pip uninstall chartinkbyvk
pip install --upgrade git+https://github.com/vkdatta/chartinkbyvk.git
```

## Usage

```bash
chartink intersect filename.txt
```

```bash
chartink union filename.txt
```

```bash
chartink all filename.txt
```
